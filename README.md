# Guardian Sleeves — Marketing Site

**Client:** Guardian Sleeves  
**Agency:** Griffith Creative  
**Stack:** Astro 5 + Tailwind CSS 4 + Vercel

## Local Development

```bash
npm install
npm run dev
```

Dev server runs at `http://localhost:4321`

## Deploy

Push to `main` — Vercel auto-deploys on every push.

Manual deploy:

```bash
vercel --prod
```

## Phase 2 (Planned)
